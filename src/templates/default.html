Mission

The Librarian is a modern front-end framework for Jcink forums. This program will negate the need to build skins directly from the Admin CP Panel. Instead, you can use this framework to build a local copy of Jcink on your computer using a local live-reloading server through Webpack. Build new skins in a fraction of the time by using and customizing many built-in modules.
